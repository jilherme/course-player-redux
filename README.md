
## Run Locally

Clone the project

```bash
  git clone https://github.com/jilherme/course-player-redux.git
```

Go to the project directory

```bash
  cd course-player-redux
```

Install dependencies

```bash
  pnpm i
```

Run the JSON server

```bash
  pnpm server
```


Start the frontend

```bash
  pnpm dev
```
